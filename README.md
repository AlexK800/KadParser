# KAD Arbitrage Case Parser

##  Описание

Асинхронный парсер для сайта https://kad.arbitr.ru/, который автоматически собирает данные по арбитражным делам.

Программа:
- Принимает список номеров дел из Excel
- Переходит на страницу каждого дела
- Извлекает данные из вкладки "Электронное дело":
  - последнюю дату
  - наименование документа
- Сохраняет результаты в базу данных

---

## Стек технологий

- Python 3.11
- Playwright (парсинг динамических сайтов)
- asyncio (асинхронная обработка)
- SQLAlchemy 2.0 (ORM)
- PostgreSQL / SQLite
- Pandas (чтение Excel)
- Docker + Docker Compose

---

## Архитектура

Проект построен по принципу разделения ответственности:

- `parser/` — логика парсинга сайта
- `db/` — работа с базой данных (модели, сессии, CRUD)
- `services/` — работа с входными данными (Excel)
- `utils/` — логирование
- `main.py` — оркестрация процессов (очередь, воркеры)

---

##  Как работает

1. Чтение номеров дел из Excel (`data/input.xlsx`)
2. Добавление задач в асинхронную очередь
3. Запуск нескольких воркеров (по умолчанию 5)
4. Каждый воркер:
   - открывает сайт
   - ищет дело
   - переходит в "Электронное дело"
   - извлекает последнюю запись
5. Сохранение данных в БД
6. Логирование результатов

---

##  Формат входных данных

Excel файл `data/input.xlsx`:

| case_number     |
|-----------------|
| А32-28873/2024  |
| А40-12345/2023  |
